## Installation

MathpixClient is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "MathpixClient", :git => 'https://github.com/Mathpix/ios-client.git', :tag => '0.0.1'
```
